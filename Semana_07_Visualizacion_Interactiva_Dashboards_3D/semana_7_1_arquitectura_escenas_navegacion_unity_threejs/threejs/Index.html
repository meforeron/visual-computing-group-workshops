✦ Créditos & Tecnologías

🏗️

Arquitectura

Sistema de escenas propio sin frameworks, usando CSS transitions y JS puro para la navegación.

🌌

Renderizado

Three.js r128 para la escena 3D de fondo y el viewport interactivo del juego.

🎮

Interacción

Raycasting de Three.js para detectar clics sobre objetos 3D. Sin librerías adicionales.

📦

Distribución

Archivo HTML único. No requiere servidor, npm ni build. Compatible con cualquier navegador moderno.

Three.js r128 HTML5 Canvas CSS Custom Properties Vanilla JS ES6+ WebGL Raycasting
