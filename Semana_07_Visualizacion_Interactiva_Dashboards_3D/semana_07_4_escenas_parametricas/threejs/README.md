# Escena Parametrica con React Three Fiber

Proyecto del taller "Escenas Parametricas - Creacion de Objetos desde Datos".

## Requisitos

- Node.js 20+

## Instalar dependencias

```bash
npm install
```

## Modo desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Lo que incluye

- Generacion de datos 3D por funcion matematica.
- Construccion de objetos con `.map()`.
- Condicionales para elegir geometria (`box`, `sphere`, `cylinder`).
- Parametrizacion de posicion, escala, rotacion y color.
- Controles en tiempo real con Leva.
