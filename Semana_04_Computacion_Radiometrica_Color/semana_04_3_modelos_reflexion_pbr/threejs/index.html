Fórmulas implementadas en GLSL

Lambert (difuso)

N = normalize(normal) L = normalize(lightPos - worldPos) diff = max(dot(N, L), 0.0) color = albedo * (ambient + diff)

Phong (especular)

R = reflect(-L, N) V = normalize(camPos - worldPos) spec = pow(max(dot(R,V),0.0), shininess) color = albedo*(ambient+diff) + spec

Blinn-Phong (half vector)

H = normalize(L + V) // half vector spec = pow(max(dot(N,H),0.0), shininess) // Más eficiente, resultado similar color = albedo*(ambient+diff) + spec
